#### 본문내에서 도메인 구하기

```javascript
var patt = /(http(s)?://)?w+(.w+)+/gi;
var result = (aa.value.match(patt));
```

#### 본문내에서 url구하기 (상기와 유사)

```javascript
var patt = /(http(s)?://)?w+(.w+).S*/gi;
```

#### 이메일 주소 체크

```javascript
 function chk(v){
  str='';
  re = new RegExp("hanmail.net","i");  
  re.test(v)?str=true:str=false;
  return str
 }
```

#### 홈페이지 주소 체크

```javascript
 function chk(v){
  str='';
  re = new RegExp("^http://","i");  
  re.test(v)?str='y':str='n';
  alert(str);
 }
```
